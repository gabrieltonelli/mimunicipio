# Mi Municipio - App Móvil Cliente

Brinda una serie de micro-servicios desde el municipio a la comunidad.

## Descripción

Aplicación móvil cliente con acceso mediante redes sociales cuyo objetivo es facilitar el acceso a servicios brindados desde el municipio.
Escaneo de código QR, Geolocalización, Botón de pánico, Trámites, Notificaciones, etc.

## Utilización

1) El usuario ingresa, por única vez, mediante su red social preferida. 

2) Puede acceder al menu principal o al escanéo de QR en la vía pública, que lo derivará a diferentes micro-servicios.


## Registro

La interacción del vecino se registra en el servidor para procesar sus solicitudes o para generar estadísticas y feedback según cada módulo de servicio.
